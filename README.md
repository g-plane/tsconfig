# tsconfig

Shareable TypeScript compiler configuration file (`tsconfig.json`) for Pig Fang's projects.

## Usage

Install it:

```
npm i -D @gplane/tsconfig
```

or

```
yarn add --dev @gplane/tsconfig
```

Then, edit the `tsconfig.json`:

```json
{
  "extends": "@gplane/tsconfig",
  "compilerOptions": {
    // Override here.
  }
}
```
